# 1.1 Agent Orchestrator

## Learning Objectives

Learn how to use Adobe Experience Platform Agents and Agent Orchestrator to: 

- Analyze purchase trends
- Identify high-propensity audiences
- Validate journey performance
- Create a new journey for the CitiSignal Fiber Max rollout

## Prerequisites

- Access to Adobe Journey Optimizer and Customer Journey Analytics 
- Familiarity with [Agent Orchestrator concepts](https://experienceleague.adobe.com/en/docs/experience-cloud-ai/experience-cloud-ai/agents/agent-orchestrator)
- Access to CitiSignal demo data: Purchases, Orders, Audiences, Journeys
